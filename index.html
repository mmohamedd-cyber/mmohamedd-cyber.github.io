Area of Rectangles – Algebra

Compute the area of each rectangle.
Hover over a question to reveal the answer + full working. (No double brackets used anywhere.)

Q1
diagram

Length = (x + 4) cm, Width = 5 cm. Find an expression for the area.

Tip: multiply the constant across the bracket.
Answer + Working
Area = length × width = (x + 4) × 5 = 5x + 20
Final: 5x + 20

Q2
expand 1 bracket

Length = 3x cm, Width = (x + 2) cm. Find the area.

No double brackets — just distribute 3x.
Answer + Working
Area = length × width = 3x × (x + 2) = 3x·x + 3x·2 = 3x² + 6x
Final: 3x² + 6x
